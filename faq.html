FAQ

Для соблюдения правил информационной безопасности срок действия сертификата ограничен. Сертификат выдается владельцу на год, по истечении которого его необходимо продлить.

Продлить сертификат можно при личном обращении Пользователя в отделение Агропромбанка , а также дистанционно – с помощью электронного заявления. Воспользоваться электронной формой заявления можно только при действующем сертификате, т.е. если срок действия сертификата еще не истек. Оформление заявления осуществляется с помощью программного обеспечения Пользователя.

Для этого Пользователю необходимо подать заявление на аннулирование сертификата открытого ключа электронной подписи в Удостоверяющий центр. Заявление может быть оформлено в электронном виде с использованиемпрограммного обеспечения Пользователя. Бумажная форма заявления принимается вотделениях Агропромбанка.

Пользователь получает официальное уведомление об аннулировании сертификата не позднее одного рабочего дня с момента подачи заявления в Удостоверяющий Центр.

Для оформления заявления на выпуск квалифицированного сертификата открытого ключа необходимы следующие документы:

для граждан ПМР – паспорт ПМР или паспорт СССР, который внесен в базу данных МВД ПМР.

Для проверки о наличии паспортных данных в базе МВД ПМР необходимо обратиться в паспортный стол по месту жительства.

Первоначально убедитесь в том, что соблюдены следующие системные требования:

  • операционная система Microsoft Windows 7 Service Pack 2 и выше,
  • установлен Microsoft .Net Framework 4.7.2 и выше (скачать),
  • установлены доступные обновления операционной системы.

Для проверки наличия доступных обновлений на Microsoft Windows 7 в меню «Пуск» выберите «Панель управления» => «Система и безопасность» => «Центр обновления Windows» и далее нажмите кнопку «Проверка обновлений».

Для проверки наличия доступных обновлений на Microsoft Windows 8 и 10 в меню «Пуск» выберите «Параметры» => «Обновление и безопасность» => «Центр обновления Windows» и далее нажмите кнопку «Проверить наличии обновлений».

Если все требования соблюдены, но тем не менее при установке или обновлении приложения возникает ошибка, сообщите нам детали проблемы. Это можно сделать в три шага:

  • нажмите кнопку «Подробно» («Details») в диалоговом окне ошибки,
  • сохраните текст ошибки в файл (блокнот),
  • отправьте текстовый файл на электронную почтуca@agroprombank.com
